# 🎬 YT-DLP Video Downloader

Simple YouTube downloader using yt-dlp (DevOps friendly project)

## 🚀 Features
- Download long videos (6+ hours)
- Best quality video + audio
- Automated script

## 🛠️ Setup
1. Install Python
2. Install yt-dlp:
   pip install yt-dlp

## ▶️ Usage
Run:
download.bat

Paste YouTube URL and download starts.

## 💡 DevOps Use Case
- Automate downloads using cron jobs
- Integrate into pipelines
- Useful for media processing workflows
